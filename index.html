How it actually measures

Most "wattage estimators" just multiply TDP × utilisation. Biddut Meter reads the real sensors when they're available.

⚡

Real GPU power

NVIDIA cards report their actual power draw over NVML. No estimation, no model lookup — the chip tells us directly.

🔥

Real CPU power

Intel RAPL (read via LibreHardwareMonitor when run as admin) gives us the package-level energy counter. AMD falls back to TDP×util.

🇧🇩

DPDC tariff aware

Choose flat or tiered (slab-aware) mode. Tiered understands which DPDC residential slab your household consumption pushes you into.

🔔

Budget alerts

Set a monthly Tk limit. A Windows toast pops the moment you cross it, then re-alerts every +Tk N over.

📊

Two views

Big dashboard for deep dives + a tiny always-on-top overlay (360×200) when you just want the running cost visible while gaming.

🏠

House calculator

New in v1.3+: also estimate the rest of your household — ACs, fans, fridges, water pumps. See the PC's share of the whole bill.

Honest answers

Is it really free? ▾

Yes — fully free, no ads, no telemetry, no upsell. The source is on GitHub. If it saves you Tk and you want to send a thank-you, the GitHub Issues tab is wide open.

Why does it ask for admin permission? ▾

To read Intel RAPL counters for accurate CPU power. Without admin it still runs — CPU power just falls back to a TDP×utilisation estimate (~±20-30% on CPU; GPU stays exact via NVML either way).

Does it work with AMD GPUs? ▾

NVIDIA NVML is the cleanest sensor path. AMD GPUs fall back to TDP × utilisation. Roughly correct for ballparking; not as exact as NVML.

Does it auto-update? ▾

Not yet — for now, watch the Releases page. Re-installing preserves all your settings and tracked history (stored in %LOCALAPPDATA%\BiddutMeter\).

Windows SmartScreen says "unknown publisher" — is that safe? ▾

Yes. The installer isn't code-signed yet (those certificates cost ~$100/yr — getting there). Click "More info" → "Run anyway", or build from source on GitHub if you'd rather see the code first.

Does it work outside Bangladesh? ▾

Yes — change the Tk/kWh rate in Settings to your local rate (or override currency in a future release). The DPDC slab logic is opt-in via the "Tiered" toggle.
